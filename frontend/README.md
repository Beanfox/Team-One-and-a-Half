# TrafficBot Frontend

Modern React + TypeScript + Vite dashboard UI for the TrafficBot project.

## Runtime (No venv)

This frontend is a **Node/NPM app only**.

- No Python virtual environment is required.
- No `venv` / `.venv` setup is used by this project.

## Setup

From the project root:

```bash
npm --prefix frontend install
```

## Run (development)

```bash
npm --prefix frontend run dev
```

Vite will print a local URL (for example `http://localhost:5173/`).

## Build (production)

```bash
npm --prefix frontend run build
```

## Lint

```bash
npm --prefix frontend run lint
```
